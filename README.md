# Engenharia de Prompt e Aplicações em IA
Este repositório é destinado para trabalhos, atividades e desafios da disciplina de mesmo nome do título. Explora técnicas e metodologias para projetar, otimizar e avaliar interações com modelos de linguagem e sistemas de IA generativa. O foco está em transformar ideias em instruções eficazes que resultem em respostas úteis, criativas e confiáveis.
