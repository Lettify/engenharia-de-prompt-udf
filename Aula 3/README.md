# Diretório de conteúdos de atividades da Aula 3

## Conclusão reflexiva
Ainda não abri o meu ponto de vista de forma completa o suficiente para entender que a resposta da IA é influenciada bastante pela tokenização organizada de um prompt. Mas sim pela forma como o prompt em si é estruturado, usando a Persona, Objetivo, Contexto, Restrição e Formato. Então minha conclusão foi criar um prompt completo, complexo e bem estruturado usando esses fundamentos a caminho do *prompt perfeito* para poder atingir o meu objetivo pessoal ao solicitar algo para algum LLM.
